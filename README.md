# Sales Dashboard Project (SQL + Power BI)

## Project Overview
This project analyzes customer sales data using SQL and visualizes insights using Power BI dashboards.

The goal was to clean, transform, and model sales data to generate meaningful business insights.

---

## Tools Used
- SQL (Data Cleaning & Analysis)
- Power BI (Data Visualization)
- Data Modeling
- DAX Measures

---

##  Key Insights
- Identified Top 10 Products by Sales
- Analyzed Monthly Sales Trends
- Measured Total Sales Performance
- Evaluated Customer Purchasing Behavior

- SQL queries file
- Power BI dashboard screenshots
- Sales data analysis visuals
